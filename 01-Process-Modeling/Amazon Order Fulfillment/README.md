## Objective
To model and document the end-to-end Amazon order fulfillment
process, covering customer ordering, order processing, inventory
validation, fulfillment, shipping, and customer notifications.

## Process Overview
This BPMN diagram represents a conceptual, end-to-end view of how
Orders are fulfilled on the Amazon platform. It includes both
Amazon Fulfilled (FBA) and Merchant Fulfilled scenarios and
highlights key decision points and parallel activities.

This process model was created to demonstrate BPMN modeling
skills and process understanding. It does not represent an
official As-Is or To-Be process.

## Key Stakeholders
- Customer
- Amazon E-commerce Platform
- Amazon Fulfillment Center (FBA)
- Third-Party Merchant
- Logistics & Delivery Partners

## Key Process Steps
- Customer places an order
- Amazon reviews and validates the order
- Inventory availability check
- Payment processing and confirmation
- Order fulfillment (Amazon or Merchant)
- Shipping, tracking, and status updates
